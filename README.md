# Power BI Sales Dashboard

📊 **Project:** Sales Performance Analysis  
🛠 **Tool:** Power BI  
📁 **Dataset:** Sales_Data.csv

---

## 🔎 Overview

This project demonstrates the use of Power BI to create an interactive Sales Performance Dashboard using sales data.

The dashboard includes:

- Total Sales KPI
- Total Profit KPI
- Sales by Region (Bar Chart)
- Sales by Product Category (Pie Chart)
- Profit Trend Over Time (Line Chart)
- Quantity Sold by Product Category (Bar Chart)

---

## 🎯 Key Insights

- Technology category generated the highest sales and quantity sold.
- Profit has varied across years with a slight drop in 2024.
- Sales are quite balanced across regions.
